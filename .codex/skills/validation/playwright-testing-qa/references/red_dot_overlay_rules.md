# Red Dot Overlay Rules

- Add a visible red circle before each scripted click.
- Marker minimum style:
- diameter: 16px,
- fill: `#d80b0b`,
- 2px white border,
- `position: fixed`,
- `z-index: 2147483647`.
- Keep markers persistent through capture so screenshots and video preserve click evidence.
- Ensure critical path clicks are always marked.
