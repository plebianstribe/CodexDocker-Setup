# Planning Applicability Contract

Use these dimensions only when they materially apply to the user objective.

## Optional Dimensions
- Multi-actor partitioning (AI-only, human-only, agentic interface).
- Human control points (confirmation gates, intervention triggers, manual overrides).
- Retrieval boundary constraints (local-first vs external sources).
- Evidence integrity depth (provenance, attribution granularity, contradiction/uncertainty handling).
- Domain-specific evidence criteria variants.

## Rule
Do not force any optional dimension into all goals. Include it only when omission would make the plan non-operational or non-verifiable.
